# Hang Megiddo — Battle of Megiddo (c. 1457 BCE)

3D real-time strategy: command **Thutmose III** against the Canaanite coalition on the plain before Megiddo.

**Engine:** Godot **4.7** · touch-first · Egyptian visual theme  

## Play

**Browser (GitHub Pages):** https://drewarrowood.github.io/hangmegiddo/

```powershell
godot --path "C:\Users\drewa\groktest\states\Gaston\megiddo_rts\godot"
```

Or double-click `launch.bat`.

## Controls

| Input | Action |
|-------|--------|
| Tap unit | Select (Egyptian) / Attack (enemy if you have selection) |
| Tap ground | Move selected |
| **⚔ ATTACK-MOVE** then ground | Advance & engage |
| Pinch / wheel | Zoom |
| Drag empty / WASD | Pan |
| RMB drag | Orbit |
| Space | Pause |

## Victory

1. Shatter Canaanite strength (~28% remaining)  
2. Hold Megiddo’s approaches ~25s with superior force  
3. Or outlast on the clock with higher remaining strength  

## Design notes

- Chariots are fast shock troops — historical edge of New Kingdom Egypt.  
- AI screens the plain, garrisons Megiddo, and counters when you press the city.  
- Same spirit as Hang Calhoun (orders, sides, victory), rebuilt as fluid 3D RTS.
